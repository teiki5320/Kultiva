# Illustrations saisonnières Kultiva

Place ici les quatre illustrations kawaii de jardin potager :

- `spring.png` — printemps 🌸
- `summer.png` — été ☀️
- `autumn.png` — automne 🍂
- `winter.png` — hiver ❄️

Format recommandé : PNG, ratio ~16:9, ~1920×1080, fond doux pastel.

Une fois les fichiers ajoutés, décommente la section `assets:` dans
`pubspec.yaml` pour les exposer au build Flutter.

Tant que les fichiers ne sont pas présents, `SeasonHeader` retombe sur un
dégradé pastel correspondant à la saison — l'appli reste entièrement
fonctionnelle.
